# Crear Issue Rápido

Crea un nuevo issue en Linear.


## Instrucciones:

1. Usar el servidor MCP de Linear para crear el issue
2. Asignarlo automáticamente al usuario actual
3. Establecer prioridad media (3) por defecto
4. Agregar etiqueta "development" si existe
5. Colocarlo en el backlog del equipo actual
6. Si estamos en una rama git, agregar el nombre de la rama en la descripción

Especificaciones adicionales provista por el usuario: "$ARGUMENTS"


Responder con el ID del issue creado y un enlace directo.
